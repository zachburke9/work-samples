Ink and Amber

Technical editorial. A deep slate ink, one warm amber accent, warm paper, and a blueprint motif held in reserve for the cartography moments. Every text pairing is annotated with a measured WCAG contrast ratio: the same discipline as the data work.

Palette

Primitive tokens. Ratios are measured, not estimated. Amber is large-text and UI only on light backgrounds; it is AAA text on ink.
